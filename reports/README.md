# Reports Directory

This folder holds generated reports and MD deliverables for Assignment 2.

- `reports/snyk` - Snyk JSON reports and analysis templates
- `reports/sonarqube` - SonarQube analysis templates and screenshots (generated externally)
- `reports/zap` - OWASP ZAP reports (HTML/XML/JSON) and analysis templates

Use the scripts in `scripts/` to generate reports and fill in the MD templates with evidence and screenshots.
